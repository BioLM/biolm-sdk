``biolm workspace``
=====================

Manage workspaces.

Usage
-----

List, show, create, or delete workspaces.

.. code-block:: bash

   biolm workspace list
   biolm workspace show WORKSPACE_ID
   biolm workspace create NAME
   biolm workspace delete WORKSPACE_ID
