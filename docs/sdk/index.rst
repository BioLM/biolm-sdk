SDK Reference
=============

Reference for the ``biolm`` Python package (``pip install biolm-sdk``). For onboarding and client usage patterns, see :doc:`../intro/sdk-overview`.

.. list-table::
   :header-rows: 1
   :widths: 28 72

   * - Module
     - Description
   * - :doc:`models`
     - High-level model interface (``biolm()``, ``Model``).
   * - :doc:`protocols`
     - Protocol YAML workflows, validation, and programmatic runs.
   * - :doc:`pipeline`
     - Multi-stage protein design pipelines with DuckDB caching.
   * - :doc:`workspaces`
     - Workspace management API.
   * - :doc:`volumes`
     - Volume storage API.
   * - :doc:`io`
     - FASTA, CSV, JSON, and PDB file helpers.
   * - :doc:`finetune`
     - XGBoost and DSM finetuning workflows.
   * - :doc:`hub`
     - biolm-hub gateway discovery and configuration.
